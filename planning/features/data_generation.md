# Data Generation

Feature: Generate synthetic e-commerce source data for Snowflake ingestion.

Deliverables:
- `python/generate_data.py`
- CSV files in `data/raw/`
- Data model documentation in `planning/PLAN.md`

Key elements:
- Customers, Products, Orders, Order Items
- realistic values for revenue, categories, payment methods
- export to CSV and JSON variants
- repeat customer and sales volume patterns
